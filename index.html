Tutoriel de tagsy

Ce tutoriel va vous permettre d'utiliser ce nouvel interface de tagsy et à créer des exercices pour j'accorde.

- Comment créer un exercice?

Entrez le nom de l'exercice.

Cochez le type d'exercice que vous souhaitez : (QCM, Identificaion, réponse courte).

Le même type d'exercice doit être utilisé pour un seul même exercice.

Vous avez la possibilités de changer le type d'exercice lors de la création de la première question avec le bouton .

- Pour un QCM :
  1. Entrez l'énoncé.
  2. Sélectionnez le mot ou groupe de mot et ajouté les balises souhaitées.
  3. Dans le tableau en dessous de l'énoncé, entrez vos propositions de réponses et cocher la/les bonne(s) réponse(s). Ajoutez chaque proposition avec le button
  4. Si vous souhaitez ajouter un texte explicatif, cocher la case et
    ajoutez votre explication.
  5. Ajoutez la question créée avec le bouton
- Pour une identification :
  1. Entrez l'énoncé.
  2. Sélectionnez le mot ou groupe de mot et ajouté les balises souhaitées.
  3. Si aucune réponse n'est présente dans la phrase on ajoute la balise
    <span class="button-choice-1">Aucun<span> avec le bouton
    dans la barre d'outils
  4. Si vous souhaitez ajouter un texte explicatif, cocher la case et
    ajoutez votre explication.
  5. Ajoutez la question créée avec le bouton
- Pour une réponse courte :
  1. Entrez l'énoncé.
  2. Sélectionnez le mot ou groupe de mot et ajouté les balises souhaitées. !!! ATTENTION !!! les balises ne fonctionnes pas comme pour le QCM voir "LES BALISES"
  3. Dans le tableau en dessous de l'énoncé, entrez vos propositions de réponses et
    cocher la/les bonne(s) réponse(s).
  4. Si vous souhaitez ajouter un texte explicatif, cocher la case et ajoutez
    votre explication.
  5. Ajoutez la question créée avec le bouton "Ajouter la question".
- Générer un exercice :

Cliquez sur le button cela vous créera un fichier .CSV avec tout votre exercice.

- Une sauvegarde automatique

Une sauvegarde automatique vous est proposée si vous souhaitez sauver votre exercice
en cours d'édition. Si vous voulez stopper la création et revenir dessus plus tard, si votre navigateur se ferme,... Par défaut, il est désactivé mais vous pouvez l'activer en cochant Sauvegarde automatique en haut de la page.

- Les balises

Les balises [%R1]... [/%R1] permettent à l'algorithme d'identifier la ou les parties de l'énoncé sur lesquelles porte la question (et doivent être masqué).

Pour les réponses courtes, contrairement au QCM le texte mis entre les balises apparaît bien dans la question. Il faut donc indiquer la question et non la réponse
Exemple : Je [%R1]être[/%R1] venu.

Par défaut, le texte entre les balises est remplacé par un "?" (Exemple).
On peut utiliser les balises [%G1]...[/%G1] pour indiquer une partie du texte à mettre en surbrillance à la place (Exemple).

Tout texte mis entre les balises [%I1]...[/%I1] sera mis en surbrillance quand on cliquera sur
la réponse correspondante en mode réponse (Exemple).

Les balises peuvent s'auto-incrémeter si on active le compteur automatique dans la barre d'outils et elles recommence à 1 pour chaque nouvelle question.

- Modification ou suppression d'une réponse en cours de création

Pour modifier une réponse en cours de création, il suffit de cliquer sur et ensuite de confirmer la modification. Une fois les modifications effectuées, cliquez à nouveau sur
le button pour valider vos modifications.

Pour supprimer une réponse en cours de création, il suffit de cliquer sur et
de confirmer la suppression de la réponse. ( !!! ATTENTION !!! une fois supprimée, il n'est pas possible de récupérer la réponse).

- Modification ou suppression d'une question créée.

Pour modifier une question créée , il suffit de cliquer sur et de confirmer la modification de la question. une fois la confirmation faite, vous pouvez modifier votre énoncé et
vous pouvez faire les mêmes étapes que la modification en cours d'édition pour le tableau, ajouter, modifier ou supprimer votre explication. Une fois toutes les modications faites (énoncé, tableau de réponses, explication), vous devez confirmer les changements avec le button

Pour supprimmer une question créée, il suffit de cliquer sur et de confirmer
la suppression de la question. (!! ATTENTION !! une fois supprimée il n'est pas possible
de récupérer la question).
